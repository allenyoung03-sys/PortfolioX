# PortfolioX

一款 iOS 个人投资组合追踪 App，支持 A 股、美股、港股多市场持仓管理。

## 功能

- **多市场支持** — 同时管理 A 股、美股、港股持仓
- **总览看板** — 组合总资产、各市场分部、个股盈亏一目了然
- **实时行情** — 自动获取股票实时报价和当日涨跌
- **汇率换算** — 美股/港股资产自动按汇率换算为人民币
- **预警通知** — 自定义涨跌幅阈值，触发时推送通知
- **AI 分析** — 集成 AI 对异动股票生成分析解读
- **数据持久化** — 使用 SwiftData 本地存储，支持离线查看

## 技术栈

- SwiftUI + SwiftData
- 新浪财经 API（A 股行情）
- 腾讯财经 API（美股/港股行情）
- ExchangeRate-API（汇率数据）

## 要求

- iOS 16.0+
- Xcode 15.0+

## 开始使用

1. 克隆项目
2. 复制 `Secrets.plist.template` 为 `Secrets.plist`，填入 API Key（如有）
3. 用 Xcode 打开 `portfoliox.xcodeproj`
4. 选择模拟器或真机，运行
